# India Health Management Information System (HMIS) Dashboard

## Project Overview

This project analyzes India's Health Management Information System (HMIS) data using Excel and Power BI.

The objective is to transform raw healthcare data into meaningful insights through data cleaning, analysis, visualization, and interactive dashboards.

## Domain

Healthcare Analytics

## Tools Used

* Microsoft Excel
* Power BI
* Power Query
* DAX

## Dataset Source

Government of India - Health Management Information System (HMIS)

## Key Features

* Data Cleaning and Transformation
* State-wise Healthcare Analysis
* District-wise Performance Analysis
* Healthcare Indicator Tracking
* Interactive Power BI Dashboard
* KPI Monitoring

## Excel Analysis

* Data Cleaning
* Duplicate Removal
* Pivot Tables
* Pivot Charts
* KPI Calculations

## Power BI Dashboard

### Page 1 - Executive Summary

* Total Records
* Total States
* Total Districts
* Healthcare KPIs

### Page 2 - State Analysis

* State-wise Comparison
* Top Performing States
* Bottom Performing States

### Page 3 - District Analysis

* District-wise Insights
* Trend Analysis

### Page 4 - Healthcare Indicators

* Indicator Performance
* Comparative Analysis

## Business Impact

This dashboard helps identify healthcare performance trends and supports data-driven decision-making for healthcare monitoring and planning.
